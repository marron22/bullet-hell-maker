# 弾幕メーカー v0.18

JSaB風の回避アクションゲーム向けに、弾幕やレーザー、危険領域をタイムライン上で作成・確認するためのブラウザツールです。

公開URL:
https://marron22.github.io/bullet-hell-maker/

## 基本的な使い方

1. 「追加」メニューから弾幕プリセット、またはパッケージを追加します。
2. タイムライン上のバーをドラッグして発生タイミングを調整します。
3. 右側の「プロパティ」で選択中の攻撃を編集します。
4. 音楽を使う場合は「音楽」タブから音源を読み込み、BPMや1小節の時間を設定します。
5. プレビューで動きを確認し、必要に応じてタイミングや軌道を調整します。
6. 「ファイル」からプロジェクト、またはUnity向けデータを書き出します。
7. Gemini などで作ったAI譜面JSONは「ファイル」→「AI譜面読み込み」から取り込めます。BPMを含むJSONでは、開始位置や時間系パラメータを拍数でも指定できます。プロンプト例は `docs/gemini-beatmap-prompt.md` にあります。

## モード

- 全体編集: 通常の編集モードです。イベントの追加、選択、タイムライン移動、プロパティ編集を行います。
- 軌跡編集: 選択中イベントの軌跡を表示しながら編集します。連射系は最初の弾を中心に表示します。
- プレビュー: プレイヤーを操作して実際の避け感を確認します。タイムライン操作も可能です。
- プレビュー中は上部の「軽量」「タイムライン」で、描画負荷と表示情報量を切り替えられます。

## 右側タブ

- イベント: タイムライン上の親イベント一覧です。パッケージで生成された子イベントは、パッケージタブに表示されます。
- パッケージ: パッケージの設定、パッケージごとの弾テクスチャ、そこから生成される実際の攻撃一覧を確認します。子イベントをダブルクリックするとプロパティを確認できます。
- プロパティ: 選択中の通常攻撃、またはパッケージ内の子攻撃を細かく編集します。
- 音楽: 音楽読み込み、音量、BPM、1小節の時間を設定します。

## パッケージ

パッケージは、複数の攻撃をまとめて生成する仕組みです。

例:

- 定点からランダム方向に弾幕を発射
- 縦横レーザーを繰り返し発生
- ボムが飛来して破裂弾を発射
- ランダムな位置に円形・四角形の危険領域を発生
- スネーク状に正方形が連なる攻撃
- 中央から全方向に回転レーザーを発射

パッケージ設定を変更すると、内部の攻撃は再生成されます。内部攻撃を直接編集したい場合は、パッケージタブの一覧から対象をダブルクリックしてプロパティを開きます。

## 書き出し

- プロジェクト書き出し: 画面の状態を再現するための保存形式です。パッケージ、生成済み攻撃、タイムライン、BPM設定、パッケージごとのプレビュー用テクスチャなどを含みます。
- Unity向け書き出し: `*.stagedata.json` と `*.bulletbuffers.json` の2つを書き出します。パッケージ自体は含めず、パッケージから生成された実際の攻撃を StageData / BulletBuffer 形式に変換します。

プロジェクトを後から再編集したい場合は、必ず「プロジェクト書き出し」を使ってください。

Unity向け書き出しでは、StageData 側の `bulletSpawners[].clipName` が BulletBuffers 側の `bulletBuffers[].name` を参照します。座標はプレビュー画面を Unity 側の左下原点、右上 `(18, 36)` のワールドとして扱い、Y軸は Unity に合わせて上向きに変換します。

通常の丸弾は Unity 側の `normal` として扱われるように `typeId: 1` / `typeName: "normal"` で出力します。レーザーは細長い画像を Unity 側の画面範囲より長く伸ばす前提で出力し、太さは `bulletSpawners[].laserWidth` で調整します。

## 難易度とプレビュー

Easy / Normal / Lunatic を切り替えながら、イベントごとの表示、弾数倍率、速度倍率、予告時間倍率を編集できます。Unity向け書き出しは現在選択中の難易度の設定を反映します。

パッケージごとの弾テクスチャはプレビュー専用です。読み込んだ画像と表示倍率はプロジェクト書き出しに保存されます。敵プレビューも画像差し替えに対応し、開始位置には出現前の予告が表示されます。

敵プレビューは、出現開始位置、出現完了位置、消滅位置、表示角度を編集できます。パッケージ内の Generated Attacks は、共通しているパラメータだけをプロパティタブで一括編集できます。

タイムラインでは複数選択中のイベントをドラッグすると、選択中のイベントをまとめて移動できます。

## キーボードショートカット

- Space: 再生 / 停止
- Ctrl + C: 選択中イベントをコピー
- Ctrl + V: コピーしたイベントを貼り付け
- Backspace: 選択中イベントを削除
- Ctrl + Z: 取り消し
- Ctrl + Shift + Z / Ctrl + Y: やり直し
- Ctrl + マウスホイール: タイムラインを拡大 / 縮小
- Shift + マウスホイール: タイムラインを縦スクロール
- 上下キー: 選択中イベントのタイムライン行を移動

プレビューモード:

- WASD / 矢印キー: プレイヤー移動
- Space: 再生 / 停止
- Shift / 右Shift: ダッシュ

## タイムライン

- イベントバーをドラッグすると開始時刻を変更できます。
- 「スナップ」をオンにすると、BPMグリッドの拍線・小節線に吸着しやすくなります。
- タイムラインの上端をドラッグすると、タイムラインの高さを変更できます。
- 右側パネルとの境界をドラッグすると、イベント・プロパティ領域の幅を変更できます。
